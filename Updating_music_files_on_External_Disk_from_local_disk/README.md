#### Welcome to USB Music Dir Updater!
##### This is a program for automative updating music dir on your USB device
##### The program detect your music directory on selected Linux partition and
##### after that, program detect music directory on selected USB storage disk(if not - create).
##### Next, program checks if new files present on your seletected Linux partition.
##### Well, if new MP3 files present - copy to your USB music directory.
##### Just make sure you have enough USB space. If not - don't worry, program will tell you)
##### If you like and care about your soundtracks, be always with updated music!

#### How to start program:
##### 1) Create venv: python3 -m venv <your_venv_name>
##### 2) Activate venv: source <your_venv_name>/bin/activate
##### 3) Install libs: pip install -r requirements.txt
##### 4) Start script: python music_updater.py
